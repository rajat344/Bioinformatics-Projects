# 🧬 Bioinformatics Projects

Bioinformatics analysis using Python and BioPython.
Built by learning real-world genomics workflows.

---

## 📁 Projects

### 1. DNA Analysis (DNA.ipynb)
- DNA to RNA transcription
- RNA to Protein translation  
- GC Content calculation
- Basic sequence operations

### 2. FASTA Gene Analysis (FASTA.ipynb)
- Reading real FASTA files
- Processing multiple genes
- Analyzing BRCA1, TP53, INS genes

### 3. NCBI Real Data Analysis (NCBI.ipynb)
- Fetching live gene data from NCBI database
- Analyzed real human BRCA1 gene (7088 bp)
- GC Content analysis
- Start codon detection
- Generated analysis reports

### 4. Data Visualization (DATA VISULISATION.ipynb)
- GC Content line plots
- BLAST results bar charts
- Phylogenetic tree construction
- Multi-panel research figures

### 5. Practice Problems (PRACTICE.SA.ipynb)
- Sequence alignment
- BLAST searches
- Mystery DNA identification

---

## 🛠️ Tools Used
- Python 3.13
- BioPython
- Matplotlib
- NCBI Database

---

## 🔬 Key Concepts Covered
- Central Dogma (DNA → RNA → Protein)
- Sequence Alignment
- BLAST Search (BLASTN, BLASTX)
- GC Content Analysis
- Phylogenetic Trees
- E-value interpretation

## 📊 Real World Applications
- Cancer gene analysis (BRCA1)
